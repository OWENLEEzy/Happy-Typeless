# Output Examples

This directory contains example HTML reports generated by Happy-Typeless.

## Files

### `Typeless_Report_Example.html`
- **Data Source**: 500 mock voice transcriptions
- **Language**: English (Default)
- **Version**: v3 (3-tab layout redesign)
- **Generated**: March 2025

### `Typeless_Report_Example_zh.html`
- **Data Source**: 500 mock voice transcriptions
- **Language**: Chinese (Simplified)
- **Version**: v3 (3-tab layout redesign)
- **Generated**: March 2025

## New v3 Layout

The v3 reports feature a redesigned 3-tab layout:

1. **Overview & Profile (概览与画像)** - Core metrics, personality profile, efficiency scores
2. **Content & Communication (内容与沟通)** - Content analysis, communication style, mental wellness, intent analysis
3. **Time & Habits (时间与习惯)** - Time trends, usage patterns, extreme comparisons

## Viewing the Examples

Open the HTML file in any modern web browser:

```bash
# macOS
open output/examples/Typeless_Report_Example.html

# Windows
start output/examples/Typeless_Report_Example.html

# Linux
xdg-open output/examples/Typeless_Report_Example.html
```

## Generating Your Own Example

```bash
# Generate English version with 500 mock records
uv run typeless analyze -m --no-open --lang en

# Generate Chinese version with 500 mock records
uv run typeless analyze -m --no-open --lang zh

# Specify custom output path
uv run typeless analyze -m -o output/examples/MyExample.html
```
